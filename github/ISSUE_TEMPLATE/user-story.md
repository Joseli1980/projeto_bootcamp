**As a** user  
**I need** to login using my email and password  
**So that** I can access my profile and personalized data.

## Acceptance Criteria
**Given** I am on the login page  
**When** I enter a valid email and password and click the "Login" button  
**Then** I should be redirected to my dashboard.

**Given** I enter an invalid password  
**When** I click the "Login" button  
**Then** I should see an error message "Invalid credentials".
